Представляет кнопку на игровом экране.

Атрибуты: x (int): X-координата кнопки. y (int): Y-координата кнопки. image (pygame.Surface): Изображение кнопки. rect (pygame.Rect): Прямоугольник, определяющий размеры и положение кнопки.
